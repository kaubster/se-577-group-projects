This page provides the ability to define a new dataset and its data definition manually or by coping then modifying an existing dataset.

  1. Text area : Dataset Name
  2. Dataset definition attribute: Name, Type ; depending on type UI shows one of the following:
    • integer : 2 Textboxes (min, max)
    • floating : 2 Textboxes (min, max)
    • enumerated 1 Textbox (space or comma separated values)
    • arbitrary : Table of Key Value pairs (col1 name, col2 value) - time permitting
  3. Existing datasets show within a table.
  4. User adds additional attributes that stack to define remainder of dataset.
  5. Button: "Save Dataset"
  6. Datasets current definiton show within a table.
  7. Response (fail or success) indicated via text message next to button "Save Dataset" button.
