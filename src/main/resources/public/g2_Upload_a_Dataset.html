This page provides the ability to define a new dataset by uploading an XLS, XLSX or CSV file.

  2. Area on the screen where user can drag a file or a button to select a file (shows file dialog):

    • file examples provided within the Upload_Test_Files folder provided within the solution folder.
    • only one file at a time
  3. Dataset created by the files added to a table.
  4. User clicks "Upload" (file uploaded)
  5. Response (fail or success) indicated via text message next to button "Upload" button.
